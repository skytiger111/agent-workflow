💡 使用方式

1. 選擇 Config — 上方「Config 檔」下拉選單，切换不同工作流設定

2. 修改設定 — 修改基本資料（名稱、路徑、Git）或 Agent 流程

3. 新增 Agent — 點「＋ 新增 Agent」，編輯 prompt 模板

4. 另存新檔 — 輸入檔名（如 config.myproject.yaml）再點「💾 儲存」

5. 啟動工作流 — 切換到「📊 儀表板」，輸入需求描述，點「▶ 啟動工作流」
